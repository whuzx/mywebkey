PNG (slower than JPEG) -> PNG (mais lento que JPEG)
JPEG is a lossy compression. It is fast and has a very good compression ratio, but the image gets noisy. PNG is much slower, the image size is larger, but it uses lossless compression. -> JPEG é um tipo compressão de imagem com perda de qualidade. É rápido e tem uma taxa de compressão muito boa, mas a imagem fica um pouco distorcida. PNG é mais lento, o tamanho da imagem é maior , mas utiliza compressão sem perdas.
Use only first framebuffer -> Use somente framebuffer primeiro
Usually Android uses double buffering: while the phone uses one of the two framebuffers to display, it prepares the new image on the other one. Then swaps them. That's an old trick to avoid flickering. On some older phones (for example on HTC G1 aka Dream) may display some parts from the first, and some parts from the second framebuffer, but in this case usually the second framebuffer doesn't contains a whole image. This option forces Webkey to use only the first framebuffer. -> Normalmente o Android usa buffer duplo: enquanto o telefone utiliza um dos dois framebuffers para mostrar na tela, ele prepara uma nova imagem sobre a outra. Então troca elas. Isso é um velho truque para evitar a imagens trêmulas. Em alguns telefones mais velhos (por exemplo, sobre HTC G1 aka Dream) pode apresentar algumas partes do primeiro e algumas partes do segundo framebuffer, mas neste caso geralmente o segundo framebuffer não contém uma imagem inteira. Esta opção força o Webkey à usar somente o primeiro framebuffer
Record images -> Gravar imagens
The phone starts to save images to a temporary folder /sdcard/webkey_TEMP/ as fast as it can. It uses the selected options (Low resolution, Horizontal, Flip, Png/Jpeg, Use only first framebuffer). After pressing Stop recording, the phone makes a zip archive from the images and sends it to the browser. -> O telefone começa a gravar as imagens para uma pasta temporária / sdcard / webkey_TEMP / o mais rápido quanto ele pode. Ele usa as opções selecionadas (Baixa resolução, Horizontal, Flip, Png/Jpeg, usa somente o primeiro framebuffer). Depois de apertar o botão Parar de gravar, o telefone cria um arquivo zip com as imagens e envia para o browse

SMS mode -> Modo SMS
If you try to send special characters, like õ, û in an SMS message, then these characters force Android to switch to unicode format, where the number of characters in a message drops from 160 to 70 in GSM networks. If SMS mode is checked, then lines with "works in SMS mode" have priority. -> Se você tentar enviar caracteres especiais como Å, Å± em uma mensagem SMS, eles forçam o Android à escolher o formato unicode, onde o número de caracteres na mensagem é cortado de 160 para 70 nas redes GSM.
