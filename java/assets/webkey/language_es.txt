Low resolution -> Baja resolución
It makes resolution half both in horizontal and vertical direction. -> Reduce la resolución a la mitad en horizontal y vertical.
JPEG is a lossy compression. It is fast and has a very good compression ratio, but the image gets noisy. PNG is much slower, the image size is larger, but it uses lossless compression. -> JPEG es una compresión con pérdidas. Es rápida y tiene un buen ratio de compresión, pero la imagen se vuelve borrosa. PNG es más lenta, el tamaño de la imagen es mayor, pero emplea una compresión sin pérdidas.
Usually Android uses double buffering: while the phone uses one of the two framebuffers to display, it prepares the new image on the other one. Then swaps them. That's an old trick to avoid flickering. On some older phones (for example on HTC G1 aka Dream) may display some parts from the first, and some parts from the second framebuffer, but in this case usually the second framebuffer doesn't contains a whole image. This option forces Webkey to use only the first framebuffer. -> Normalmente Android emplea un buffer de imagen doble: mientras el teléfono emplea una de las imágenes para mostrarla, prepara la nueva imagen. Entonces las intercambia. Este en un viejo truco para evitar el parpadeo. En algunos teléfonos antiguos (Como el HTC G1 o Dream) pueden mostrarse algunas partes de la primera y algunas de la segunda, pero en este caso la segunda no suele contiener la imagen completa. Esta opción fuerza a Webkey a emplear solo la primera.
Record images -> Grabar imágenes
The phone starts to save images to a temporary folder /sdcard/webkey_TEMP/ as fast as it can. It uses the selected options (Low resolution, Horizontal, Flip, Png/Jpeg, Use only first framebuffer). After pressing Stop recording, the phone makes a zip archive from the images and sends it to the browser. -> El teléfono comienza a guardar las imágenes en la carpeta temporal /sdcard/webkey_TEMP/ tan rápido como puede. Emplea las opciones seleccionadas (Baja resolución, Pantalla horizontal, Voltear, PNG/Jpeg, Emplear solo la primera imagen del buffer). Tras presionar Detener la grabación, el teléfono crea un archivo zip de las imágenes y las envía al navegador.
Typing -> Escritura
The large text area is for inject keys to the phone. You can do it immediately if "Send while typing" is checked, or you can send all the written text by pressing the button Send text. The injection of every non-English character may reload your phone's screen at the first time. The button DEL injects Back. -> El área (cuadro) de texto es para introducir caracteres en el teléfono. Puedes hacerlo de forma inmediata si "Enviar mientras se escribe" está activado o puedes enviar todo el texto escrito presionando el botón "Enviar texto". La introducción de todo caracter no inglés puede recargar la pantalla del teléfono la primera vez. El botón DEL introduce el comando Volver.

SMS mode -> Modo SMS
If you try to send special characters, like ő, ű in an SMS message, then these characters force Android to switch to unicode format, where the number of characters in a message drops from 160 to 70 in GSM networks. If SMS mode is checked, then lines with "works in SMS mode" have priority. -> Si tratas de enviar caracteres especiales, como ő, ű en un SMS, estos caracteres fuerzan a Android a cambiar a formato unicode, donde el número de caracteres para un mensaje pasa de 160 a 70 en las redes GSM. Si el modo SMS está activo, entonces las líneas con "Trabajar en modo SMS" tinen prioridad.
